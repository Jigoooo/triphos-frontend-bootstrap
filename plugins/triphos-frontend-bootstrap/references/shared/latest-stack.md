# Triphos Latest Stack Reference

This file captures the approved official reference set for the bootstrap.

## Stable defaults

- React 19 + React Compiler
- Vite 8
- TanStack Router / Query / Virtual
- Zustand
- Vitest 4
- `@jigoooo/api-client`
- `framer-motion`
- `@floating-ui/react`
- TypeScript 6.0
