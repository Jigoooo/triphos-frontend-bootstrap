# Language Policy

- 사용자에게 보이는 설명, 질문, 진행 상황, 오류 요약, 최종 보고는 사용자의 마지막 실질 요청 언어를 따른다.
- 한국어가 포함된 요청, 한국어/영어 혼합 요청, 언어 판단이 애매한 요청은 한국어로 답한다.
- 사용자가 명시적으로 특정 언어를 요청하면 그 언어를 우선한다.
- 코드, 경로, 명령어, 패키지명, API 이름, `SSR`/`SPA`/`app`/`app-ssr` 같은 기술 토큰은 원문을 유지한다.
- 선택지를 물을 때도 설명 문장은 사용자 언어를 따르고, 선택지 라벨이 기술 토큰이면 원문을 유지한다.
