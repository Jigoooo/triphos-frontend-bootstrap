# DatePicker Mobile Sheet And Font Tune

## Goal

Remove the lingering date-picker trigger focus ring after open/close interactions, switch the picker to a bottom-sheet presentation on small screens, and raise the most obviously undersized template text without changing the global rem base.

## Scope

- `shared/ui/date-picker` trigger/content behavior
- home route text sizing
- toggle button text sizing
- visual baseline refresh if `/starter` shifts
